## Microsoft Azure Academy

### Partnership Overview

The ICDFA is an official **Microsoft Azure Academy Partner**, providing ACICP trainees with comprehensive cloud security training through Microsoft's official curriculum and hands-on labs.

### Benefits for ACICP Trainees

| Benefit | Details |
|:--------|:--------|
| **Azure Credits** | $200 USD monthly credits per fellow for cloud labs and projects |
| **Microsoft Learn** | Full access to Microsoft Learn for Educator platform with 500+ courses |
| **Official Curriculum** | AZ-500 (Azure Security Engineer) and AZ-900 (Azure Fundamentals) courses |
| **Certification Exams** | Discounted exam vouchers for Microsoft Azure certifications |
| **Hands-On Labs** | 50+ pre-built lab environments for cloud security training |
| **Support** | Microsoft technical support for lab infrastructure and tools |

### ACICP Integration

Microsoft Azure is integrated into the following ACICP courses:

- **ACICP 601 – Cloud Security Fundamentals:** Azure security architecture, identity management, and data protection
- **ACICP 602 – Cloud Security Automation:** Azure DevSecOps, infrastructure as code (IaC), and automated security controls
- **ACICP 603 – Virtual Infrastructure:** Azure virtual machines, networking, and storage management

### Certification Pathways

ACICP trainees are encouraged to pursue the following Microsoft certifications:

1. **Azure Fundamentals (AZ-900):** Foundational cloud knowledge
2. **Azure Administrator (AZ-104):** Cloud infrastructure management
3. **Azure Security Engineer (AZ-500):** Cloud security specialization (recommended for ACICP graduates)

### Getting Started

1. **Enrollment:** Upon admission to ACICP, you will receive Microsoft Learn credentials
2. **Credit Activation:** Azure credits are activated on Day 1 of the programme
3. **Support:** Microsoft technical support is available 24/7 for lab issues
4. **Training:** Instructors will guide you through Azure labs during ACICP 601–602

### Microsoft Contact

**Microsoft Azure Academy**  
Email: academy@microsoft.com  
Website: https://www.microsoft.com/en-us/education/institutions/azure-academy

## Network Development Group (NDG) NETLAB+

### Partnership Overview

The ICDFA partners with **Network Development Group (NDG)** to provide ACICP trainees with access to **NETLAB+**, an advanced network simulation and hands-on lab platform. NETLAB+ is particularly valuable for ICS/SCADA security training.

### Benefits for ACICP Trainees

| Benefit | Details |
|:--------|:--------|
| **24/7 Lab Access** | Unlimited access to NETLAB+ environment 24 hours a day, 7 days a week |
| **Pre-Built Scenarios** | 200+ pre-configured lab scenarios including ICS/SCADA systems |
| **Real Equipment Simulation** | Realistic simulation of Cisco, Juniper, and industrial equipment |
| **ICS/SCADA Labs** | Specialized labs for critical infrastructure and industrial control systems |
| **Collaborative Labs** | Multi-user labs for team-based exercises and simulations |
| **Performance Tracking** | Detailed analytics and progress tracking for all lab activities |
| **Technical Support** | NDG technical support for lab infrastructure and troubleshooting |

### ACICP Integration

NETLAB+ is the primary platform for the following ACICP courses:

- **ACICP 201 – Network Security Engineering:** Network design and firewall configuration labs
- **ACICP 202 – Security Operations & Threat Monitoring:** Network monitoring and threat detection labs
- **ACICP 301 – ICS & SCADA Systems:** PLC programming and SCADA system labs
- **ACICP 302 – ICS Network Security & Attacks:** ICS network security and attack simulation labs
- **ACICP 303 – Industrial Exploitation & Pivoting:** Advanced ICS exploitation and network pivoting labs
- **ACICP 603 – Virtual Infrastructure (Proxmox):** Virtual machine and container deployment labs

### Lab Scenarios Available

NETLAB+ provides access to the following types of lab scenarios:

- **Networking Fundamentals:** Routing, switching, and network design
- **Network Security:** Firewalls, VPNs, and intrusion detection
- **ICS/SCADA:** Power systems, water treatment, manufacturing
- **Cloud Networking:** Multi-cloud network design and security
- **Incident Response:** Simulated security incidents and response procedures
- **Compliance:** Network design for regulatory compliance (NERC, NIST)

### Getting Started

1. **Enrollment:** Upon admission to ACICP, you will receive NETLAB+ credentials
2. **Lab Access:** NETLAB+ is available on Day 1 of the programme
3. **Training:** Instructors will provide orientation and guidance on using NETLAB+
4. **Support:** NDG technical support is available for lab issues and troubleshooting

### NETLAB+ Contact

**Network Development Group (NDG)**  
Email: support@netdevgroup.com  
Website: https://www.netdevgroup.com/

## How to Access Partner Resources

### Step 1: Admission to ACICP

Upon acceptance into the ACICP, you will receive:

- ACICP Student Portal credentials
- Welcome email with partner resource information
- Partner resource activation guide

### Step 2: Activate Partner Accounts

On **Day 1 of the ACICP**, you will:

1. Log into the ACICP Student Portal
2. Click "Activate Partner Resources"
3. Follow the activation wizard for each partner
4. Receive credentials for Microsoft Azure, AWS Academy, Cisco Academy, and NETLAB+

### Step 3: Access Resources

Once activated, you can access partner resources through:

- **Microsoft Azure:** https://portal.azure.com (using your academy credentials)
- **AWS Academy:** https://www.awsacademy.com (using your academy account)
- **Cisco Academy:** https://www.netacad.com (using your academy account)
- **NETLAB+:** https://netlab.netdevgroup.com (using your NETLAB+ credentials)

### Step 4: Get Support

For technical issues with partner resources:

1. **First:** Contact your ACICP instructor or teaching assistant
2. **Second:** Use partner technical support (contact information provided upon activation)
3. **Third:** Contact ICDFA Student Services for escalation
