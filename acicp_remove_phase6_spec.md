# ACICP Follow-Up Redesign: Remove Phase 6 / ACICP 604

## Requested Change
Remove **Phase 6: Virtual Infrastructure** and **ACICP 604 – Advanced Virtual Infrastructure** from the published ACICP curriculum, then adjust the programme so it still ends with a clear final phase.

## Resulting Curriculum Shape
The published advanced programme should remain a **24-week / 6-month** structure, but it should now be presented as **6 phases** instead of 7.

## Revised Phase Layout
| New Phase | Focus Area | Duration | Courses |
|:--|:--|:--|:--|
| Phase 1 | Core Systems & Security | 4 weeks | ACICP 101 |
| Phase 2 | Network & Security Operations | 3 weeks | ACICP 201-202 |
| Phase 3 | Critical Infrastructure (ICS) | 4 weeks | ACICP 301-303 |
| Phase 4 | Offensive Cyber Operations & Firewall Engineering | 4 weeks | ACICP 401-403 |
| Phase 5 | Cloud Security | 2 weeks | ACICP 601 |
| Phase 6 | Digital Forensics & Final Simulation | 7 weeks | ACICP 701-702 |

## Summary Totals
- Published course sections decrease from **13** to **12**.
- Published lab count decreases from **123** to **105**.
- Phase count decreases from **7** to **6**.
- Programme duration remains **24 weeks (6 months)** by expanding the final phase to absorb the removed 3-week virtual infrastructure block.

## Document Update Rules
1. Remove the table-of-contents link for Phase 6: Virtual Infrastructure.
2. Remove the full ACICP 604 section and lab table from the main curriculum.
3. Rename the old Phase 7 section to **Phase 6: Digital Forensics & Final Simulation**.
4. Replace references to a seven-phase structure with a six-phase structure.
5. Replace references to 13 published course sections / 123 labs with **12 published course sections / 105 labs**.
6. Remove mentions of virtual infrastructure from high-level programme summaries unless used only as a general lab capability statement.
7. Update partnership mappings so ACICP 604 no longer appears in integration examples.
